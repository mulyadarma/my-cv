About Me

Let me tell you a few things...

Perkenalkan..

Saya Sandy Mulyadarma, saya pria warga negara Indonesia sudah menikah telah dikaruniai satu putri, Saya lahir di Bandung tanggal 07 April 1984 dengan pendidikan terakhir S1 Jurusan Akuntansi di Universitas Padjadjaran Bandung.
Ketertarikan saya terhadap dunia website bermula dari kegiatan browsing yang tidak dapat saya lepaskan dalam kehidupan sehari hari sehingga saya sering mengamati bagaimana tampilan sebuah website, informasi apa yang ditawarkan dan interaksi setiap web dengan penggunanya dapat mempengaruhi saya dalam menggunakan website tersebut. Hal inilah yang membuat saya sangat tertarik bahwa ternyata tampilan website, cara sebuah website mempermudah penggunannya dalam memenuhi kebutuhan, dapat sangat membentuk image sebuah brand yang melekat dalam website tersebut dan website menurut saya representasi sebuah perusahaan.
Setelah mengetahui ternyata membuat tampilan website tidak harus lulusan Teknik Informasi dan mengikuti pelatihan website harus mengeluarkan biaya yang cukup besar akhirnya saya memutuskan untuk mempelajari membuat website melalui video tutorial di Youtube. Semoga sedikit ilmu yang saya ketahui menggenai HTML, CSS dan Javascript dalam membuat website CV saya membuka kesempatan saya untuk dapat memberi kontribusi di perusahaan yang Bapak/Ibu pimpin.

Atas segenap perhatiannya saya mengucapkan Terimakasih.
